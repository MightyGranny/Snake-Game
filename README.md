# Snake Game 🐍

A classic Snake Game built with Python and Pygame.

## Features

- Smooth snake movement
- Food spawning
- Snake growth
- Score counter
- Wall collision
- Self collision
- Quit and Restart options

## Technologies Used

- Python 3.13
- Pygame 2.6.1

## Installation

```bash
pip install -r requirements.txt
python snake.py
```
